DataSpider 使用指南

一、项目简介

本项目为基于 Django 框架开发的网页爬虫系统，支持用户注册、登录后提交网站链接进行网页内容抓取，包括文字与图片，并可在网页端系统中查询、管理相关数据。

二、开发环境

  • 操作系统：Windows 10+、macOS、Linux
  • Python：推荐 ≥ 3.11（示例为 3.12.6）
  • MySQL：推荐 ≥ 8.0（示例为 8.0.41）
  • 依赖安装：pip install -r requirements.txt

三、数据库初始化

  1. 确保已启动 MySQL，并已创建数据库
  2. 在 mysite/ 目录下创建 .env 文件，内容如下：
DB_NAME=your_database
DB_USER=root
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=3306

然后执行迁移命令：

python manage.py makemigrations crawls
python manage.py migrate

四、运行项目

python manage.py runserver

打开浏览器访问：http://127.0.0.1:8000

五、主要功能

  • 爬取任务创建与调度
  • 网站、网页、图片、文本内容浏览
  • 关键词搜索
  • 用户注册、登录与权限管理
  • 数据删除（限拥有者/管理员）
